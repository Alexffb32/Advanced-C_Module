Projeto 1 Luz RGB Automática

O objetivo do projeto é montar o esquema acima e com o devido código a a led deve dar luz e alternar entre varias cores automaticamente durante um espaço de tempo.

Projeto 2 Sistema Semáforo

O objetivo do projeto é montar o esquema acima e com o devido código é fazer um sistema de semáforo onde as luzes das cores vermelho, amarelo e verde irao ligar ou desligar automaticamnete fazendo uma sequencia.

Projeto 3 Leds com Joystick

O objetivo do projeto é montar o esquema acima e com o devido código os pinos de led devem acender tendo em conta a direção a que o jpystick é apontado, em tempo real.
