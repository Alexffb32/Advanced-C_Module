Modulo C Avançado Mais Sobre C

C é uma linguagem de programação de uso geral. Foi criado na década de 1970 por Dennis Ritchie e continua sendo amplamente utilizado e influente. Por design, os recursos do C refletem claramente as capacidades das CPUs alvo.

Neste modulo abordamos mais expecificamnete a Linguagem C e trabalhamos mais com a Linguagem C++ tanto na programação dos arduinos que montamos como no projeto de detecção de caras para livestrams e mettings.
